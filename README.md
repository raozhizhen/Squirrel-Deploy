# Squirrel-Deploy
Squirrel 简体中文带 emoji 表情和词库扩展的配置文件。

[先下载输入法](http://rime.im/)

再下载此库，将库里的 Rime 文件夹替换 ~/Library/Rime/ 文件夹。

control + ~ 切换输入法为 朙月拼音・简化字 。

control + option + ` 重新部署一下。

完成。

验证

输入 “upan” 、 “ip” 看看，是不是有 U 盘和 iPad

输入 “表情” 看看是不是有表情。
